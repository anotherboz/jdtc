Les évêques

Les évêques sont nommés par le Pape. Ils sont soit évêques diocésains (en charge d’un diocèse) soit titulaires (ils ont une fonction particulière).

Les évêques “dirigent le domaine”, ils gèrent les biens et dirigent les prêtres. Leur couleur est le violet.
