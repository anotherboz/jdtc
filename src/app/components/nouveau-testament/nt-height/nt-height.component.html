Pour résumer

Matthieu : Homme - Son évangile commence par la généalogie de Jésus

Marc : Lion - Le début de l’évangile évoque la voix dans le désert

Luc : Taureau - Son évangile commence dans le temple où l’on sacrifie des taureaux

Jean : Aigle - Est celui qui prend le plus de hauteur

L’origine de ces attributions provient d’une vision d'Ezéchiel (ancien testament) et repris dans un passage de l’apocalypse : « Le premier vivant est comme un lion ; le deuxième vivant est comme un jeune taureau ; le troisième vivant a comme un visage d’homme ; le quatrième vivant est comme un aigle en plein vol. »
